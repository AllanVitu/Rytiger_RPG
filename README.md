# Rytiger RPG ⚔️

Bienvenue dans **Rytiger RPG**, un jeu de rôle au tour par tour (style Heroic Fantasy / Dark Fantasy Abyssale) optimisé pour le format Desktop ! Collectionnez de nombreux héros, lancez-vous dans des combats épiques et progressez au travers d'innombrables zones et donjons pour purifier l'Abysse Éternelle.

## 🎮 Jouer en ligne

Vous pouvez jouer immédiatement à Rytiger RPG directement depuis votre navigateur en cliquant sur le lien ci-dessous :

👉 **[Jouer à Rytiger RPG](https://allanvitu.github.io/Rytiger_RPG/)**

---

## 🛠️ Installation en local (IDE)

Si vous souhaitez télécharger le projet, explorer le code ou le modifier dans votre IDE (Visual Studio Code, Cursor, etc.), voici comment procéder :

### Prérequis
Assurez-vous d'avoir installé [Node.js](https://nodejs.org/) sur votre machine.

### Étapes d'installation

1. **Ouvrez votre terminal et clonez le projet (ou téléchargez-le)**
   ```bash
   git clone https://github.com/allanvitu/Rytiger_RPG.git
   cd Rytiger_RPG
   ```

2. **Installez les dépendances**
   ```bash
   npm install
   ```

3. **Lancez le serveur de développement**
   ```bash
   npm run dev
   ```

4. **Accédez au jeu**
   Ouvrez le lien fourni par le terminal (généralement `http://localhost:3000` ou `http://localhost:5173`) dans votre navigateur de choix.

---

### 🗡️ Technologies utilisées
- **Vite** pour le serveur de développement
- Vanilla JS, HTML, et Vanilla CSS
- Architecture avec router personnalisé.
